## Introduction

`@univerjs/sheets-zen-mode-editor` enhances the immersive editing experience for cells in sheets.

## Usage

### Installation

```shell
# Using npm
npm install @univerjs/sheets-zen-editor

# Using pnpm
pnpm add @univerjs/sheets-zen-editor
```
